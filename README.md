**Streamline** (formerly AI Lesson Planner) is a sophisticated automation suite designed to handle the heavy lifting of educational planning. It leverages advanced web orchestration and local intelligence to generate, format, and deploy lesson resources across various educational platforms.

## Features
- **Auto-Planning:** Logic-driven creation of weekly lesson schedules based on subject requirements.
- **Resource Generation:** Automatic creation and formatting of PDF, DOCX, and Text-based teaching aids.
- **Portal Sync:** (Planned) Integration for automatically spreading resources across school portals using Playwright.
- **Systemd Integration:** Built for reliable, background execution on Linux/Raspberry Pi environments.
